| 原则 | 说明 |

## 五、数据库选型

### 5.1 MySQL vs SQLite

| 对比项 | MySQL | SQLite | 选择 |
|--------|-------|--------|------|
| 并发支持 | 高 | 低 | MySQL |
| 数据量 | 1000+ | 500+ | MySQL |
| 部署复杂度 | 中等 | 简单 | 视情况 |
| 事务支持 | 完整ACID | 有限 | MySQL |
| 维护成本 | 中等 | 低 | - |

### 5.2 MySQL 配置要求

| 配置项 | 最低要求 | 推荐配置 |
|--------|----------|----------|
| CPU | 1核 | 2核 |
| 内存 | 512MB | 2GB |
| 磁盘 | 10GB | 50GB |
| 版本 | 5.7+ | 8.0+ |

## 七、环境配置

### 7.1 Node.js 版本管理

推荐使用 nvm 管理 Node.js 版本：

```bash
# 安装 nvm (Windows)
scoop install nvm

# 安装 Node.js 18
nvm install 18
nvm use 18
```

### 7.2 环境变量配置

```env
# .env.example (后端)

# 服务器配置
PORT=3000
NODE_ENV=development

# 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_NAME=taskbar
DB_USER=root
DB_PASSWORD=your_password

# Session配置
SESSION_SECRET=your_session_secret_here_min_32_chars
SESSION_MAX_AGE=86400000
```

### 7.3 前端环境配置

```env
# .env.production (前端)
VITE_API_BASE_URL=http://localhost:3000
VITE_APP_TITLE=学院工作任务栏系统
```
