# 04-数据库设计输入.md

---

## 一、数据库设计原则

| 原则 | 说明 |
|------|------|
| 规范化 | 遵循第三范式，减少数据冗余 |
| 完整性 | 主键、外键、唯一约束保证数据正确性 |
| 扩展性 | 预留字段，便于功能扩展 |
| 命名规范 | 小写下划线命名，见名知意 |

## 三、数据表设计

### 3.1 用户表 (users)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| user_id | INT | PK, AUTO_INCREMENT | 用户ID |
| username | VARCHAR(50) | UNIQUE, NOT NULL | 用户名 |
| password | VARCHAR(255) | NOT NULL | 密码（加密） |
| salt | VARCHAR(50) | NOT NULL | 盐值 |
| role | ENUM('admin','teacher','leader') | NOT NULL, DEFAULT 'teacher' | 角色 |
| staff_id | INT | FK(staff.staff_id) | 关联人员ID |
| status | TINYINT | NOT NULL, DEFAULT 1 | 状态：0禁用1正常 |
| login_attempts | INT | DEFAULT 0 | 登录失败次数 |
| lock_until | DATETIME | NULL | 锁定截止时间 |
| last_login | DATETIME | NULL | 最后登录时间 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

```sql
CREATE TABLE users (
    user_id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) NOT NULL UNIQUE,
    password VARCHAR(255) NOT NULL,
    salt VARCHAR(50) NOT NULL,
    role ENUM('admin','teacher','leader') NOT NULL DEFAULT 'teacher',
    staff_id INT,
    status TINYINT NOT NULL DEFAULT 1,
    login_attempts INT DEFAULT 0,
    lock_until DATETIME NULL,
    last_login DATETIME NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (staff_id) REFERENCES staff(staff_id) ON DELETE SET NULL
);
```

### 3.2 人员表 (staff)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| staff_id | INT | PK, AUTO_INCREMENT | 人员ID |
| staff_code | VARCHAR(20) | UNIQUE, NOT NULL | 工号 |
| name | VARCHAR(50) | NOT NULL | 姓名 |
| gender | ENUM('male','female') | NULL | 性别 |
| department | VARCHAR(50) | NOT NULL | 部门 |
| position | VARCHAR(50) | NULL | 职位 |
| phone | VARCHAR(20) | NULL | 手机号 |
| email | VARCHAR(100) | NULL | 邮箱 |
| status | ENUM('active','disabled','left') | NOT NULL DEFAULT 'active' | 状态 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

```sql
CREATE TABLE staff (
    staff_id INT PRIMARY KEY AUTO_INCREMENT,
    staff_code VARCHAR(20) NOT NULL UNIQUE,
    name VARCHAR(50) NOT NULL,
    gender ENUM('male','female') NULL,
    department VARCHAR(50) NOT NULL,
    position VARCHAR(50) NULL,
    phone VARCHAR(20) NULL,
    email VARCHAR(100) NULL,
    status ENUM('active','disabled','left') NOT NULL DEFAULT 'active',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 3.3 任务表 (tasks)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| task_id | INT | PK, AUTO_INCREMENT | 任务ID |
| task_no | VARCHAR(30) | UNIQUE, NOT NULL | 任务编号 |
| task_name | VARCHAR(100) | NOT NULL | 任务名称 |
| period_id | INT | FK(periods.period_id) | 所属周期 |
| owner_id | INT | FK(staff.staff_id), NOT NULL | 负责人 |
| description | TEXT | NULL | 任务描述 |
| priority | ENUM('high','medium','low') | NOT NULL DEFAULT 'medium' | 优先级 |
| category | VARCHAR(50) | NOT NULL | 任务分类（应用层校验） |
| status | ENUM('pending','in_progress','completed','delayed','cancelled') | NOT NULL DEFAULT 'pending' | 状态 |
| progress | INT | DEFAULT 0 | 完成进度 0-100 |
| start_date | DATE | NOT NULL | 开始日期 |
| end_date | DATE | NOT NULL | 截止日期 |
| remarks | TEXT | NULL | 备注 |
| created_by | INT | FK(users.user_id) | 创建人 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

```sql
CREATE TABLE tasks (
    task_id INT PRIMARY KEY AUTO_INCREMENT,
    task_no VARCHAR(30) NOT NULL UNIQUE,
    task_name VARCHAR(100) NOT NULL,
    period_id INT,
    owner_id INT NOT NULL,
    description TEXT NULL,
    priority ENUM('high','medium','low') NOT NULL DEFAULT 'medium',
    category VARCHAR(50) NOT NULL,
    status ENUM('pending','in_progress','completed','delayed','cancelled') NOT NULL DEFAULT 'pending',
    progress INT DEFAULT 0,
    start_date DATE NOT NULL,
    end_date DATE NOT NULL,
    remarks TEXT NULL,
    created_by INT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (period_id) REFERENCES periods(period_id) ON DELETE SET NULL,
    FOREIGN KEY (owner_id) REFERENCES staff(staff_id),
    FOREIGN KEY (created_by) REFERENCES users(user_id)
);
```

### 3.4 任务协助人表 (task_collaborators)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | ID |
| task_id | INT | FK(tasks.task_id), NOT NULL | 任务ID |
| staff_id | INT | FK(staff.staff_id), NOT NULL | 协助人员ID |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

```sql
CREATE TABLE task_collaborators (
    id INT PRIMARY KEY AUTO_INCREMENT,
    task_id INT NOT NULL,
    staff_id INT NOT NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (task_id) REFERENCES tasks(task_id) ON DELETE CASCADE,
    FOREIGN KEY (staff_id) REFERENCES staff(staff_id),
    UNIQUE KEY uk_task_staff (task_id, staff_id)
);
```

### 3.5 待办事项表 (todos)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| todo_id | INT | PK, AUTO_INCREMENT | 待办ID |
| todo_name | VARCHAR(100) | NOT NULL | 待办名称 |
| task_id | INT | FK(tasks.task_id) | 关联任务 |
| executor_id | INT | FK(staff.staff_id), NOT NULL | 执行人 |
| priority | ENUM('urgent','important','normal','low') | DEFAULT 'normal' | 优先级 |
| status | TINYINT | NOT NULL DEFAULT 0 | 状态：0未完成1已完成 |
| deadline | DATETIME | NULL | 截止时间 |
| reminder_time | DATETIME | NULL | 提醒时间 |
| completed_at | DATETIME | NULL | 完成时间 |
| remarks | TEXT | NULL | 备注 |
| sort_order | INT | DEFAULT 0 | 排序 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

```sql
CREATE TABLE todos (
    todo_id INT PRIMARY KEY AUTO_INCREMENT,
    todo_name VARCHAR(100) NOT NULL,
    task_id INT,
    executor_id INT NOT NULL,
    priority ENUM('urgent','important','normal','low') DEFAULT 'normal',
    status TINYINT NOT NULL DEFAULT 0,
    deadline DATETIME NULL,
    reminder_time DATETIME NULL,
    completed_at DATETIME NULL,
    remarks TEXT NULL,
    sort_order INT DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (task_id) REFERENCES tasks(task_id) ON DELETE SET NULL,
    FOREIGN KEY (executor_id) REFERENCES staff(staff_id)
);
```

### 3.6 会议表 (meetings)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| meeting_id | INT | PK, AUTO_INCREMENT | 会议ID |
| meeting_no | VARCHAR(30) | UNIQUE, NOT NULL | 会议编号 |
| meeting_name | VARCHAR(100) | NOT NULL | 会议名称 |
| meeting_type | ENUM('all','department','party','teaching','other') | NOT NULL | 会议类型 |
| host_id | INT | FK(staff.staff_id), NOT NULL | 主持人 |
| start_time | DATETIME | NOT NULL | 开始时间 |
| end_time | DATETIME | NOT NULL | 结束时间 |
| location | VARCHAR(100) | NOT NULL | 会议地点 |
| agenda | TEXT | NULL | 会议议题 |
| reminder_setting | ENUM('15min','30min','1hour','1day','none') | NOT NULL DEFAULT '30min' | 提醒设置 |
| status | ENUM('upcoming','ongoing','ended','cancelled') | NOT NULL DEFAULT 'upcoming' | 状态 |
| minutes | TEXT | NULL | 会议纪要 |
| created_by | INT | FK(users.user_id) | 创建人 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

```sql
CREATE TABLE meetings (
    meeting_id INT PRIMARY KEY AUTO_INCREMENT,
    meeting_no VARCHAR(30) NOT NULL UNIQUE,
    meeting_name VARCHAR(100) NOT NULL,
    meeting_type ENUM('all','department','party','teaching','other') NOT NULL,
    host_id INT NOT NULL,
    start_time DATETIME NOT NULL,
    end_time DATETIME NOT NULL,
    location VARCHAR(100) NOT NULL,
    agenda TEXT NULL,
    reminder_setting ENUM('15min','30min','1hour','1day','none') NOT NULL DEFAULT '30min',
    status ENUM('upcoming','ongoing','ended','cancelled') NOT NULL DEFAULT 'upcoming',
    minutes TEXT NULL,
    created_by INT,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (host_id) REFERENCES staff(staff_id),
    FOREIGN KEY (created_by) REFERENCES users(user_id)
);
```

### 3.7 参会人员表 (meeting_participants)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | ID |
| meeting_id | INT | FK(meetings.meeting_id), NOT NULL | 会议ID |
| staff_id | INT | FK(staff.staff_id), NOT NULL | 人员ID |
| confirmed | TINYINT | DEFAULT NULL | 确认状态：0未确认1确认2拒绝 |
| confirmed_at | DATETIME | NULL | 确认时间 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

```sql
CREATE TABLE meeting_participants (
    id INT PRIMARY KEY AUTO_INCREMENT,
    meeting_id INT NOT NULL,
    staff_id INT NOT NULL,
    confirmed TINYINT DEFAULT NULL,
    confirmed_at DATETIME NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (meeting_id) REFERENCES meetings(meeting_id) ON DELETE CASCADE,
    FOREIGN KEY (staff_id) REFERENCES staff(staff_id),
    UNIQUE KEY uk_meeting_staff (meeting_id, staff_id)
);
```

### 3.8 周期表 (periods)

| 字段 | 类型 | 约束 | 说明 |

| end_date | DATE | NOT NULL | 结束日期 |

| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

```sql
CREATE TABLE periods (
    period_id INT PRIMARY KEY AUTO_INCREMENT,
    period_name VARCHAR(50) NOT NULL,
    start_date DATE NOT NULL,
    end_date DATE NOT NULL,
    semester VARCHAR(50) NULL,
    sort_order INT DEFAULT 0,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
);
```

### 3.9 操作日志表 (operation_logs)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| log_id | INT | PK, AUTO_INCREMENT | 日志ID |
| user_id | INT | FK(users.user_id) | 操作人 |
| action | VARCHAR(50) | NOT NULL | 操作类型 |
| target_type | VARCHAR(50) | NOT NULL | 操作对象类型 |
| target_id | INT | NOT NULL | 操作对象ID |
| detail | JSON | NULL | 详细信息 |
| ip_address | VARCHAR(50) | NULL | IP地址 |

```sql
CREATE TABLE operation_logs (
    log_id INT PRIMARY KEY AUTO_INCREMENT,
    user_id INT,
    action VARCHAR(50) NOT NULL,
    target_type VARCHAR(50) NOT NULL,
    target_id INT NOT NULL,
    detail JSON NULL,
    ip_address VARCHAR(50) NULL,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(user_id) ON DELETE SET NULL
);
```

### 3.11 待办共享表 (todo_shares)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | ID |
| todo_id | INT | FK(todos.todo_id), NOT NULL | 待办ID |
| share_to_staff_id | INT | FK(staff.staff_id), NOT NULL | 共享给的人员ID |
| permission | ENUM('view','edit') | DEFAULT 'view' | 权限：查看/编辑 |
| created_at | DATETIME | DEFAULT CURRENT_TIMESTAMP | 创建时间 |

```sql
CREATE TABLE todo_shares (
    id INT PRIMARY KEY AUTO_INCREMENT,
    todo_id INT NOT NULL,
    share_to_staff_id INT NOT NULL,

);
```

## 四、索引设计

### 4.1 索引清单

| 表名 | 索引名 | 字段 | 类型 | 说明 |
|------|--------|------|------|------|
| users | idx_username | username | 普通 | 用户名查询 |
| users | idx_role_status | role, status | 复合 | 角色状态筛选 |
| staff | idx_staff_code | staff_code | 唯一 | 工号查询 |
| staff | idx_name | name | 普通 | 姓名搜索 |
| staff | idx_status | status | 普通 | 状态筛选 |
| tasks | idx_task_no | task_no | 唯一 | 任务编号查询 |
| tasks | idx_owner | owner_id | 普通 | 负责人筛选 |
| tasks | idx_period | period_id | 普通 | 周期筛选 |
| tasks | idx_status | status | 普通 | 状态筛选 |
| tasks | idx_category | category | 普通 | 分类筛选 |
| tasks | idx_end_date | end_date | 普通 | 截止日期排序 |
| todos | idx_executor | executor_id | 普通 | 执行人筛选 |
| todos | idx_task | task_id | 普通 | 关联任务查询 |
| todos | idx_status | status | 普通 | 状态筛选 |
| meetings | idx_meeting_no | meeting_no | 唯一 | 会议编号查询 |
| meetings | idx_host | host_id | 普通 | 主持人筛选 |
| meetings | idx_start_time | start_time | 普通 | 时间排序 |
| meetings | idx_status | status | 普通 | 状态筛选 |
| categories | idx_category_name | category_name | 唯一 | 分类名称查询 |
| categories | idx_sort_order | sort_order | 普通 | 排序 |
| todo_shares | idx_todo | todo_id | 普通 | 待办筛选 |
| todo_shares | idx_share_to | share_to_staff_id | 普通 | 共享人员筛选 |

### 4.2 索引SQL

```sql
-- 用户表索引
CREATE INDEX idx_username ON users(username);
CREATE INDEX idx_role_status ON users(role, status);

-- 人员表索引
CREATE UNIQUE INDEX idx_staff_code ON staff(staff_code);
CREATE INDEX idx_name ON staff(name);
CREATE INDEX idx_status ON staff(status);

-- 任务表索引
CREATE UNIQUE INDEX idx_task_no ON tasks(task_no);
CREATE INDEX idx_owner ON tasks(owner_id);
CREATE INDEX idx_period ON tasks(period_id);
CREATE INDEX idx_status ON tasks(status);
CREATE INDEX idx_category ON tasks(category);
CREATE INDEX idx_end_date ON tasks(end_date);

-- 待办表索引
CREATE INDEX idx_executor ON todos(executor_id);
CREATE INDEX idx_task ON todos(task_id);
CREATE INDEX idx_todo_status ON todos(status);

-- 会议表索引
CREATE UNIQUE INDEX idx_meeting_no ON meetings(meeting_no);
CREATE INDEX idx_host ON meetings(host_id);
CREATE INDEX idx_start_time ON meetings(start_time);
CREATE INDEX idx_meeting_status ON meetings(status);

-- 分类表索引
CREATE INDEX idx_category_name ON categories(category_name);
CREATE INDEX idx_sort_order ON categories(sort_order);

-- 待办共享表索引
CREATE INDEX idx_todo ON todo_shares(todo_id);
CREATE INDEX idx_share_to ON todo_shares(share_to_staff_id);
```
